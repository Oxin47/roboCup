Rules - main

  • Tournament format is play-off.
  • Games are played on zagram. Tournament grid is displayed here.
  • Game settings are: 3 minutes for the first turn, 1 minute starting time, sec/turn, unrated, 1-cross. Games are started automatically by the server.
  • Time between rounds (time to rest) is set to minutes.
  • Tournament timezone is UTC+4. (It's now ).

Rules - additional

  • A random player is picked up as a winner in two cases: either game length has exceeded (hours:minutes) or players have equal dot count at the end of game
  • By registering or taking part in the tournament you agree to the tournament rules. Tournament is offered to the participants "as is".
  • In case of questions you should write to the zagram chat or the inner tournament chat.
  • Tournament organizators are Oleg Anokhin, Vanya Geyko and Vasya Novikov. Visualization, grid forming and game assignment are done by the server.
