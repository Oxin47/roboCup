Язык

Чтобы сменить язык на русский:

firefox -> tools -> preferences -> content -> languages (снизу) > choose -> russian (ru)
opera -> tools -> preferences -> language -> ru_RU (русский язык)
chromium -> гаечный ключ -> settings -> under the hood -> web content -> languages -> russian (русский)

Język

Żeby zmienić język na polski:

firefox -> tools -> preferences -> content -> languages (in the bottom) > choose -> polish (pl)
opera -> tools -> preferences -> language -> pl (Polish)
chromium -> settings -> under the hood -> web content -> languages -> polish
