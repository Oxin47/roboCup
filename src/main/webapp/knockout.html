Specific rules

  • Tournament format is play-off.
  • Game settings are: 3 minutes for the first turn, 1 minute starting time, sec/turn, unrated, 1-cross. Games are started automatically by the server.
  • Time between rounds (time to rest) is set to minutes.
  • Each tour opponents are assigned ramdomly (by the server).
  • A random player is picked up as a winner in two cases: either game length has exceeds (hours:minutes) or players get equal dot count at the end of game.
